# Thyroidectomy Risk Calculator

## Introduction
This project is a Streamlit application designed to help assess the risk associated with thyroidectomy procedures. It provides an intuitive interface for inputting patient data and receiving risk assessments based on predefined algorithms.

## Prerequisites
- Python 3.7 or later
- Streamlit library
- Additional libraries (e.g., pandas, numpy, scikit-learn) required for the application.

## Installation Steps
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/surgeonspirit-bharath/thyroidectomy-risk-calculator.git
   cd thyroidectomy-risk-calculator
   ```

2. **Install the Required Libraries**:
   You can install the necessary libraries using pip:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Application
1. Open a terminal and navigate to the project directory (if not already there).
2. Run the following command to start the Streamlit app:
   ```bash
   streamlit run app.py
   ```
3. Your web browser should automatically open to `http://localhost:8501`, where you can interact with the application.

## Usage
Follow the on-screen instructions to input patient data and obtain a risk assessment.
